# KinologBot-
# Dog Trainer Booking Bot (Telegram)

Telegram-бот для кинолога: приём заявок от клиентов, подтверждение/отклонение, запись на занятия по слотам времени, эксклюзивная занятость слота (1 клиент = 1 время), просмотр расписания кинологом, отзывы и автоматические уведомления.

---

## Что делает бот
- Показывает **правила** и запускает **регистрацию клиента** (имя/телефон/инфо о собаке).
- Отправляет заявку кинологу на **модерацию** (одобрить/отклонить).
- Даёт клиенту возможность **записаться на занятия** по дням недели и времени.
- Гарантирует правило: **один временной слот = один клиент**.
- Позволяет кинологу:
  - управлять **активностью слотов** (вкл/выкл),
  - видеть **расписание на сегодня / на неделю** (кто и во сколько записан),
  - вручную корректировать занятия клиента.
- Принимает **отзывы**.
- Отправляет **уведомления**:
  - кинологу — расписание на завтра в 21:30,
  - клиенту — напоминание за 24 часа до занятия.

---

## Для кого
- **Кинолог / школа дрессировки**, которым нужно системно вести клиентов и расписание.
- **Клиенты кинолога**, которым удобно записываться в Telegram без звонков и переписок.

---

## Какие проблемы решает
- Убирает хаос в переписке: заявки и записи проходят через понятный сценарий.
- Снижает нагрузку на кинолога: автоматизированная регистрация и статус заявки.
- Исключает конфликтные записи: **нельзя двум клиентам занять один и тот же слот**.
- Даёт прозрачность расписания: кинолог видит **кто/когда** записан на сегодня и на неделю.
- Уменьшает число “забытых занятий”: напоминания за 24 часа + ежедневное сводное расписание для кинолога.

---

## Основной функционал

### Для клиента
- **Онбординг с правилами**: пользователь подтверждает согласие кнопкой.
- **Регистрация**: имя/фамилия, телефон, информация о собаке.
- **Статус заявки**: “на рассмотрении / одобрен”.
- **Запись на занятия**:
  - выбор дня недели → выбор времени;
  - визуальные статусы слотов:  
    - `✅` — слот выбран этим клиентом  
    - `⛔` — слот занят другим клиентом  
    - пусто — слот свободен
- **Моё расписание**: отображение занятий по дням.
- **Отзывы**: отправка отзыва.

### Для кинолога (админа)
- **Новые клиенты**: просмотр заявок, одобрение/отклонение.
- **Все клиенты**: список одобренных клиентов + удаление при необходимости.
- **Слоты (вкл/выкл)**: управление доступными временами.
- **Изменить занятия клиента**: ручное назначение/снятие занятий.
- **Расписание на сегодня**: список “время → клиент/свободно”.
- **Расписание на неделю**: расписание на 7 дней вперёд.
- **Отзывы**: просмотр последних отзывов.
- **Правила**: редактирование текста правил.

---

## Стек и архитектура
- **Python 3.11+**
- **aiogram 3.x** (FSM, Router, polling)
- **SQLite** (хранение клиентов, слотов, записей, правил, логов уведомлений)
- **asyncio scheduler** (циклическая отправка уведомлений по времени)
- Деплой: VPS + `systemd` (рекомендуемый вариант для 24/7)

> Примечание: в текущей версии используется **polling**. При необходимости можно адаптировать под **webhooks** (например, для PaaS/серверless окружений).
